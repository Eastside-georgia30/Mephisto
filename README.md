## 📖 About Mephisto

Mephisto serves as a security tool for WordPress. It identifies weaknesses in your website plugins and core files. Security professionals use this scanner to find specific bugs known as CVEs. Once Mephisto finds a path into a site, it helps you understand how attackers might reach your data. You then use this information to fix the holes and keep your WordPress installation safe from outside threats.

## ⚙️ System Requirements

Before you start, make sure your computer meets these requirements:

*   Windows 10 or Windows 11.
*   At least 4GB of RAM.
*   An active internet connection to scan remote sites.
*   A user account with administrative rights to run the scanner.

## 🛡️ Usage Guidelines

Use this tool only on websites you own or have explicit permission to test. Unauthorized scanning of third-party websites may violate local laws and terms of service agreements. Treat the results as information to help you protect your digital assets. Patch your plugins immediately after you identify a vulnerability.

*   **Vulnerability Detection:** Automates the search for known CVEs within thousands of separate plugins.

*   **Report Generation:** Saves a list of found weaknesses in a text file for your review.
*   **Batch Scanning:** Allows you to input a list of links to scan multiple websites in one session.

## ❓ Frequently Asked Questions

**Does this tool work on Mac or Linux?**
The current version is built specifically for Windows. Use a virtual machine to run it on other systems.

**Is it safe to run?**
The tool performs harmless checks to identify weaknesses. It does not damage your files unless you choose to run an exploitation function.

**What do I do if I find a bug?**
Update the plugin or theme that contains the weakness. If the developer has no update, remove the plugin from your WordPress site.

**Does it require a database?**
No. The tool runs directly from your files without needing an external database setup.
